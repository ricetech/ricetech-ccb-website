Motherboard & RAM

Motherboard

One of the largest components of a computer. It contains the chipset, CPU power delivery (VRM, or Voltage Regulator Modules), sockets, and internal and rear I/O (Input/Output) ports. It contains sockets for nearly all of the components, including the CPU socket, RAM slots, SATA connectors for hard drives, SSDs and CD/DVD/Blu-Ray drives, PCIe slots for expansion cards such as sound cards, video cards, extra port cards such as USB and FireWire, headers for case components such as the case fans, power button and indicator lights, and several power connectors.

ASUS ROG Crosshair VI Hero ATX AM4

Specifications

  • Motherboard size: ATX
  • Chipset: AMD X370
  • PCIe slots: 2x 3.0 x16, 1x 2.0 x4, 3x 2.0 x1
  • SATA ports: 8 SATA III supporting RAID (Redundant Array of Independent Disks) 0/1/10
  • USB 3.1 Ports: 10 (1 Type-C)
  • Front panel USB headers: 3
  • Multi-GPU technology: nVidia 2-Way SLI, AMD 3-Way CrossFireX
  • RAM supported: 4x DDR4 2133/2400 MHz
  • Price: $329.99

ASUS has always been known for producing extremely high quality motherboards, and this one is no exception. The Crosshair VI Hero is a standard ATX (Advanced Technology eXtended) sized board with all of the features expected from a reliable company such as ASUS. It features an extremely high quality sound card along with a DAC (digital to analog converter) built into the board, several fan headers specially designated for water coolers, their extremely well-known UEFI BIOS (Unified Extensible Firmware Interface Basic Input Output System), their also extremely well-known CPU power delivery system, several ASUS exclusive builder-friendly features such as Q-connectors, Q-PCIe slots and Q-DIMM (Dual Inline Memory Module) slots, ASUS Aura RGB board lighting along with several RGB LED (Light Emitting Diode) strip headers, and all of the latest connectivity, such as USB (Universal Serial Bus) 3.1 Gen2, M.2 SSDs and WiFi Cards, 2 PCIe 3.0 x16 (Peripheral Component Interconnect Express Generation 3) slots to allow for a future installation of a second graphics card in nVidia SLI (Scalable Link Interface), Intel Gigabit LAN (Local Area Network), and an extremely large number of USB 2.0/3.0 ports. The proven reliability and large number of features for the price make this motherboard a very good choice for the build.

RAM (Random Access Memory)

Short-term memory for storing open programs and files in use by the computer. Attaches to the motherboard. Data is lost if power is lost. Much faster than a hard drive, but slower than the CPU cache. The current standard is DDR4 (Double Data Rate 4th Generation). Average capacity today is 16GB. Typically found in dual or quad channel (2 or 4) configurations.

Kingston HyperX Fury 32 GB (4x8GB) DDR4-2133

Specifications

  • Confiuration: 4x8GB
  • Form Factor: 288-pin DIMM
  • Technology: DDR4 SDRAM
  • Speed: 2133 MHz
  • Voltage: 1.2 V
  • Price: $322.99

In the market for RAM (Random Access Memory), Kingston has made a name for themselves as a reliable producer of RAM. I have personally used their HyperX memory in both my own current gaming computer, as well as a friend's gaming computer, and it has proven to be very reliable memory. The choice of a 4x8GB configuration over a 2x16GB configuration for the memory is mostly for aesthetic purposes, as I see filled memory slots as more visually appealing. As the processor only supports Dual Channel and not Quad Channel, there is no performance difference whatsoever between the two configurations. The other reason for using this particular set of memory is due to radiator clearance. The Corsair 460X does not have a lot of clearance near the top if a radiator is installed, requiring the use of low profile memory such as this set to avoid having the radiator or fans strike the memory. This, combined with the reliability of Kingston and the rather sleek black heatsink were the reasons I had for choosing this memory.
