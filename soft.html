Software

OS (Operating System)

The most important piece of software on any computer (apart from the BIOS). The OS will usually contain a GUI (Graphical User Interface) that allows the user to interact with the computer using peripherals such as the keyboard, mouse and monitor. The OS is the backbone of the other programs that are run on the computer, and assists in managing the computer itself.

Microsoft Windows 10 Home

No computer is complete without an OS (operating system). For years now, Microsoft Windows has been the industry standard consumer OS. For this build, the best choice is the newest iteration of the OS, Windows 10 Home. While the Pro version does contain extra features, they are mostly related to IT (Information Technology) and provide no benefit to a gamer or content creator. For this reason, and for the new features (including Game Mode, which improves gaming performance) included in the new Creator's Update, Windows 10 Home was selected for this build.
